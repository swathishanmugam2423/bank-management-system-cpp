# 🏦 Banking Management System

A simple Banking Management System developed using C++.

## 📌 Features
- Create New Account
- Deposit Money
- Withdraw Money
- Check Balance
- Display Account Details
- Exit System

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)

## ▶️ How to Run
1. Clone this repository.
2. Open the project in any C++ IDE (Code::Blocks, VS Code, Dev C++, etc.).
3. Compile and run the source file.
